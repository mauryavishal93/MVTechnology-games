# MV-Labs Technology — Games & Projects

Official static website for **MV-Labs Technology**, showcasing our company, games, and other projects.

## What’s included

- **About** — Who we are and what we stand for  
- **Games** — Find My Puppy & Tidal Warfare with direct Google Play links  
- **Other projects** — Mobile apps, prototypes, consulting  
- **Contact** — How to get in touch  

## Run locally

Open `index.html` in a browser, or use a local server:

```bash
# Python 3
python3 -m http.server 8000

# Node (npx)
npx serve .
```

Then visit `http://localhost:8000`.

## Host on GitHub Pages

1. Push this repo to GitHub.  
2. **Settings → Pages** → Source: **Deploy from a branch**.  
3. Branch: `main` (or `master`), folder: **/ (root)**.  
4. Save. Your site will be at `https://<username>.github.io/MVTechnology-games/` (or your repo name).

## Customize

- **Contact email**: Edit the `mailto:` link in the Contact section in `index.html` (currently set to contact@mvlabstechnology.com).  
- **Copy & projects**: Edit text in `index.html` and add more game or project cards as needed.
